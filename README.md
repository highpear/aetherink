# AetherInk

A lightweight desktop overlay drawing app for Windows and macOS.

## Goals

- Draw freely over the screen
- Support transparent overlay canvas
- Clear and simple whiteboard-style annotation
- Future click-through mode for presentations and meetings

## Current Features

- Black line drawing with mouse drag
- Adjustable pen color
- Adjustable pen width
- Undo last stroke
- Clear canvas button
- White / transparent canvas background toggle

## Tech Stack

- Rust
- egui / eframe
